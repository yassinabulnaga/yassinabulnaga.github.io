Reflow Oven Controller — ELEC291/ELEC292

Abstract

A precision reflow oven controller built on the N76E003 microcontroller and programmed entirely in assembly. The system automates the reflow soldering process using a K-type thermocouple (with OP07 signal conditioning) and an SSR to regulate a toaster oven. It features an LCD UI, passcode lock, live serial logging with a Python dashboard, and an automated door mechanism for rapid cooldown.


Figure 1: Hardware–software integration overview.

Goals

  • Closed-loop temperature control following configurable reflow profiles.
  • ±3 °C temperature accuracy across the full thermal cycle.
  • Robust, assembly-only firmware with clear FSM stages and fault handling.
  • Operator safety and usability: LCD UI, passcode lock, and automated cooldown door.
  • Real-time data logging and visualization via Python strip chart.

System Design

The controller measures oven temperature via a K-type thermocouple and amplifies/conditions the signal using an OP07 op-amp before sampling with the N76E003 ADC. A solid-state relay modulates heater power; the process is governed by an FSM that steps through Preheat, Soak, Reflow, Cool Down, Done, and Error. An LCD and pushbuttons provide a user interface, while UART streaming supports logging/visualization on a host PC.

Hardware Modules

  • Temperature Sensing: K-type thermocouple → OP07 gain/offset stage → MCU ADC.
  • Power Control: SSR driving a toaster oven element (duty-cycle / burst control).
  • User I/O: 16×2 LCD and pushbuttons for profile selection and status.
  • Door Mechanism: 12 V DC motor opens door at cycle end to accelerate cooldown.
  • Audio: PWM melody (Super Mario theme) on completion.

Software Modules

  • Finite State Machine: Rest → Ramp to Soak → Soak → Ramp to Reflow → Reflow → Cool Down → Done (→ Error on faults).
  • Control Loop: ADC sampling, profile tracking, heater drive (SSR) with thermal safeguards.
  • UI & Security: LCD updates, passcode lock, button debouncing.
  • Telemetry: UART streaming for logging and Python strip-chart plotting.


Figure 4: FSM governing the reflow process.

Results

  • Accuracy: Maintained within ±3 °C versus a calibrated Fluke 45 across the cycle.
  • Profile Tracking: Achieved JEDEC-style preheat/soak/reflow ramps with stable peak.
  • Usability: Clear LCD status, audible completion, and passcode protection.
  • Cooldown: Automated door significantly reduced cool-down time and improved safety.
