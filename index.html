16-bit RISC CPU on FPGA

Verilog RTL on DE1-SoC with modular ALU, register file, datapath, instruction/data memory, and memory-mapped I/O.

Overview

Designed a clean, modular CPU: separated control FSM and datapath; implemented arithmetic/logic ops, branches, and memory operations. Mapped board switches/LEDs as I/O for fast verification.

Highlights

  • ALU ops: add/sub/and/or/xor/shifts/compare
  • Datapath + control FSM with clear interfaces
  • Memory-mapped I/O, instruction & data RAM
  • ModelSim testbenches; Quartus synthesis

Tech Stack

Verilog, Quartus, ModelSim, DE1-SoC
