Featured Project

FlashAttention FPGA Accelerator

A tiled FlashAttention kernel targeting the DE1-SoC (Cyclone V), designed to compute scaled dot-product attention within on-chip SRAM to eliminate HBM bandwidth bottlenecks. Architecture features an 8×8 systolic PE array with FP16/fixed-point arithmetic, DMA-driven memory subsystem, and fused softmax-multiply operators — all verified in SystemVerilog simulation before FPGA deployment.

Systolic Array GEMM Accelerator

16×16 systolic PE array implemented in SystemVerilog, achieving 4.45 GMAC/s at 50 MHz on the DE1-SoC. Verified with directed and random testbenches in ModelSim; synthesized and timed in Quartus.

16-Bit RISC CPU

Modular CPU in Verilog with separate datapath and control FSM. Supports ALU ops, branches, memory-mapped I/O, and multi-cycle memory access. Verified in ModelSim and deployed on DE1-SoC via Quartus.
