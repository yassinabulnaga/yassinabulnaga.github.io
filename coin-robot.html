Coin Retrieval Robot

Abstract

A dual-mode robot designed to retrieve metallic coins either autonomously or through manual control. In manual mode, the robot is operated via a wireless PS2 remote. In autonomous mode, it executes a finite state machine that governs the search, detection, pickup, and return processes.

Goals

  • Develop a robot capable of locating and retrieving metallic coins.
  • Implement both manual and autonomous operation modes.
  • Enable wireless communication between the robot and a handheld remote.
  • Provide real-time feedback via LCD and audio signals.

System Design

The system is divided into sensing, actuation, control, and communication modules. The robot integrates STM32 (onboard) and PIC32 (remote) microcontrollers, wireless modules, and custom circuitry for detection and actuation.

Hardware Modules

  • Colpitts Oscillator – detects coins via frequency shift.
  • Electromagnet + Servo – retrieves and drops coins.
  • Perimeter Sensors – prevent escape from the arena.
  • LCD Display – shows coin count and system status.
  • DC Motors + PWM – drive and maneuver the robot.


Hardware block diagram

Software Modules

  • Finite State Machine – manages autonomous search and retrieval cycle.
  • Interrupt Service Routines – ensure responsive motor and sensor control.
  • UART Communication – facilitates data exchange with the remote.


Software block diagram

Remote Controller

Built around a PIC32MX130F064B, the handheld controller integrates a PS2 interface, LCD display, speaker, and JDY-40 wireless link for real-time control and telemetry.

Results

  • Achieved 90%+ coin retrieval success across 50+ trials.
  • Reliable perimeter detection above ~2.4 V threshold.
  • Consistent wireless communication at 9600 baud using JDY-40 modules.
